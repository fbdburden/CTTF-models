# CTTF-models repository

2026-01-15

This repository provides the different models used within the chemical and toxin task force (CTTF) in the framework for WHO's Foodborne Disease Burden Epidemiology Reference Group (FERG) towards the global burden of foodborne disease. The repository contains models used to go from exposure data to incidence/PAF for five different chemicals (aflatoxin B1, aflatoxin M1, arsenic, cadmium and methylmercury).



The different models are outlined in *World Health Organization Estimates of the Global, Regional, and National Disease Burden of Nine Foodborne Chemicals, 2000-21: An Updated Data Synthesis*, submitted to Lancet Global Health (https://doi.org/10.1016/j.langlo.2026.103986).
